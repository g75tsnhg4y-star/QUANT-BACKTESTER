# QUANT-BACKTESTER
Algorithmic Strategy & Risk Engine | Python, Streamlit, Pandas, YFinance

Architected a full-stack quantitative backtesting engine in Python to evaluate multi-regime alpha signals against continuous institutional position-sizing algorithms.

Engineered dynamic capital allocation models featuring dual-mode toggles between Volatility Scaling (Risk Parity) and optimal Kelly Fraction logic.

Designed and integrated a "Black Swan Radar," calculating rolling 4th-moment mathematical kurtosis to systematically override leverage parameters and limit tail-risk drawdowns.

Deployed the backtester via Streamlit, implementing Plotly to visualize interactive equity curves, rolling win rates, transaction-cost impacts, and underwater profiles.
